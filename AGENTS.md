## 2. CONTEXT-AWARE APEX TECH STACKS (LATE 2025)
**Directives:** This repository, `ActionsFlow-Event-Driven-Automation-GitHub-Template`, focuses on event-driven automation via Actionsflow and GitHub Actions.

*   **PRIMARY SCENARIO: WORKFLOW AUTOMATION / DEVOPS (YAML & GitHub Actions Ecosystem)**
    *   **Stack:** This project leverages **GitHub Actions** and **Actionsflow** for event-driven automation. The core definition language is **YAML**, and custom logic is implemented in **TypeScript 5.x**.
    *   **Architecture:** Adheres to an **Event-Driven Architecture (EDA)** pattern, with a focus on modular, reusable workflows and actions. Emphasis on clearly defined triggers, inputs, and outputs for seamless orchestration.
    *   **Validation & Linting:** Utilizes **Biome** for formatting and linting of TypeScript code, and `yamllint` for YAML files.
    *   **Testing:** Implements workflow validation, dry-runs where applicable, and **Vitest** for unit and integration tests of custom TypeScript logic.
    *   **Security:** Prioritizes secure coding practices for custom actions, adherence to the principle of least privilege for GitHub tokens, robust secrets management, and regular auditing of workflow permissions.
